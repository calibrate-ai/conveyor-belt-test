# Conveyor Belt Test

End-to-end pipeline test for the Calibrate agent system.

## Structure
- `backend/` — API endpoints (Forge's domain)
- `frontend/` — UI components (Anvil's domain)
- `tests/` — Test suite (Gauntlet validates)

## Workflow
Cards created in Trello → In Progress → agent builds → PR → Crucible reviews → Gauntlet QAs → Done
